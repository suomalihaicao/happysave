## 📊 当前架构速览

### 技术栈
- **框架:** Next.js 16 + React 19
- **UI:** Ant Design 5 + TailwindCSS
- **数据库:** SQLite(开发) / TiDB(生产) / PostgreSQL(可选)
- **部署:** Vercel
- **监控:** Sentry + Vercel Analytics

### 数据流
```
用户请求 → Vercel Edge → Middleware(限流/鉴权)
                       → Next.js SSR/ISR
                       → Cache Layer (SWR 5min/30min)
                       → DB (SQLite/TiDB/PG)
                       → 响应 + CDN 缓存
```

### 安全层
- Middleware: API 限流 (60次/分钟)
- Cookie 鉴权: hs_admin (Secure+HttpOnly+SameSite)
- 安全头: X-Frame-Options, CSP, etc.
- CRON: 3种认证方式

---

## 📝 工作原则

1. **先修后建** — 白屏/崩溃优先级最高
2. **类型安全** — 禁止 any，接口先行
3. **测试驱动** — 关键路径必须有测试
4. **性能预算** — 每次改动不能恶化性能
5. **安全默认** — 默认拒绝，显式允许
